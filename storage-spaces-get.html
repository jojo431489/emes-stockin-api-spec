儲位基本資料

GET ERP → MES
storage.spaces.get

從 ERP 同步儲位基本資料至 MES 的 CMSNL 資料表。使用複合主鍵 (warehouse_no + storage_spaces_no)。

特殊處理

DELETE 後 INSERT 模式
此 API 採用「先刪除再新增」模式，同步時會先刪除該倉庫下所有儲位，再重新 INSERT 新資料。
