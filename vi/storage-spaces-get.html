Xu ly dac biet

Che do DELETE truoc INSERT
API nay ap dung che do "xoa truoc them moi", khi dong bo se xoa tat ca vi tri trong kho do truoc, sau do INSERT du lieu moi.

SQL cap nhat co so du lieu

Xoa du lieu cu

-- Xoa tat ca vi tri trong kho do
DELETE CMSNL WHERE NL001 = :warehouse_no

INSERT (Them moi vi tri)

INSERT INTO CMSNL (NL001, NL002, NL003)
VALUES (
    :NL001,      -- Ma kho
    :NL002,      -- Ma vi tri
    :NL003       -- Ten vi tri
)
