Thoi diem kich hoat

Nhac nho quan trong
Huy phieu bao cong chi goi ERP khi PRODUCTION_REPORTID co gia tri. Neu khi xuat tram ERP chua tra ve so phieu bao cong (TIMEOUT hoac that bai), khi huy se khong the thong bao ERP khoi phuc.

Cac tinh huong kich hoat

Tinh huong 1: Huy xuat tram binh thuong

Nguoi dung nhan "Huy xuat tram" trong MES, he thong truy van so phieu bao cong sau do goi ERP huy phieu bao cong do.

Tinh huong 2: Huy xuat tram theo lo

Huy nhieu ban ghi xuat tram theo lo, he thong se goi ERP huy tung phieu bao cong tuong ung.

Tinh huong 3: Huy lam lai

Huy xuat tram tai tram lam lai cung se kich hoat quy trinh huy phieu bao cong.

Tham so yeu cau

Truong Ten truong ERP Kieu Bat buoc Mo ta
Ma cong ty enterprise_no String Bat buoc Ma doanh nghiep
Ma nha may site_no String Bat buoc Ma nha may
So phieu bao cong report_no String Bat buoc So phieu bao cong can huy (Loai-So)
Dinh dang so phieu bao cong
Dinh dang so phieu bao cong la PDTYPE-PDNO, vi du: 10-20260319001. Gia tri nay lay tu SFT_OP_REALRUN.PRODUCTION_REPORTID hoac SFT_WS_RUN.WR024.

  "enterprise_no": "COMP01",
  "site_no": "SITE01",

}

Logic xoa MES

Sau khi huy phieu bao cong thanh cong, MES se xoa so phieu bao cong trong cac ban ghi lien quan:

Logic xoa SQL

-- Xoa so phieu bao cong trong SFT_OP_REALRUN
UPDATE SFT_OP_REALRUN
SET PRODUCTION_REPORTID = '',
    PRODUCTION_REPORTSN = ''
WHERE ID = :KEYID
  AND ERP_OPSEQ = :ERP_OPSEQ
  AND SEQUENCE = :SEQUENCE

-- Xoa so phieu bao cong trong SFT_WS_RUN
UPDATE SFT_WS_RUN
SET WR024 = '',
    WR025 = ''
WHERE (dieu kien ban ghi nhap xuat tram tuong ung)

-- Xoa tieu de va chi tiet phieu bao cong (tuy cau hinh he thong)
DELETE FROM SFT_SFCTE WHERE TE001 = :PDTYPE AND TE002 = :PDNO
DELETE FROM SFT_SFCTD WHERE TD001 = :PDTYPE AND TD002 = :PDNO

Logic phan loai
Khi huy xuat tram, he thong phan loai API huy can goi dua tren SFT_TRANSORDER_LINE.TRANSORDERTYPE:

• TRANSORDERTYPE = 59 → outsourcingProcess (Gia cong ngoai)
• TRANSORDERTYPE = 58 → woScrapDataProcess (Phe pham)
• Khac → cancelStockIn hoac cancelProductionForm

Xu ly loi

Tinh huong loi Phan hoi ERP Cach xu ly MES
Phieu bao cong da xac nhan code=500, Phieu bao cong da xac nhan Hien thi thong bao loi, can xu ly thu cong
Phieu bao cong khong ton tai code=500, Phieu bao cong khong ton tai Xoa so phieu bao cong MES, tiep tuc quy trinh huy
Het thoi gian ket noi SocketTimeoutException Ghi LOG, can huy bo sung thu cong

Rui ro Race Condition

Luu y quan trong
Tuong tu API tao phieu bao cong, huy phieu bao cong cung co rui ro Race Condition:

Tinh huong: Huy ngay sau khi xuat tram, nhung ERP chua tra ve so phieu bao cong
Ket qua: PRODUCTION_REPORTID trong, khong the goi ERP huy
Hau qua: MES da huy xuat tram, nhung phieu bao cong ERP van ton tai

De xuat xu ly:
  1. Cho ERP tra ve thanh cong truoc khi cho phep huy sau xuat tram
  2. Dinh ky doi chieu trang thai phieu bao cong giua MES va ERP
  3. Cung cap chuc nang huy bo sung thu cong
