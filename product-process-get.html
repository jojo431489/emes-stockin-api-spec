產品途程資料

GET ERP → MES
product.process.get

從 ERP 同步產品途程（BOM）資料至 MES。包含途程表頭 (SFT_BOMME) 和途程表身 (SFT_BOMMF)。

基本資訊

API 名稱
product.process.get
方法名稱
upBomCROSS
目標資料表
SFT_BOMME (頭) / SFT_BOMMF (身)
JSON 節點
process_data
主鍵
ME001 + ME002
程式位置
RegularESB.java:2614

回應欄位

HEAD 途程表頭 (SFT_BOMME)

ERP 欄位MES 欄位說明
process_itemME001製程品號
process_codeME002製程代碼
process_nameME003製程名稱
descriptionME004製程描述

LINE 途程表身 (SFT_BOMMF)

ERP 欄位MES 欄位說明
process_seqMF003製程序號
operation_noMF004製程編號 (ERP_OPID)
workstation_noMF005工作站編號 (ERP_WSID)
std_timeMF006標準工時

資料庫更新 SQL

途程表頭 SFT_BOMME（先刪後新增）

-- 刪除舊資料
DELETE FROM SFT_BOMME WHERE ME001 = :ME001 AND ME002 = :ME002

-- 新增表頭
INSERT INTO SFT_BOMME (
    ME001, ME002, ME003, ME004, ME007,
    MES001, MES004, CREATER, CREATE_DATE, FLAG
) VALUES (
    :ME001, :ME002, :ME003, :ME004, :ME007,
    'N', 'N', :userid, N'{當前時間}', 0
)

途程表身 SFT_BOMMF（先刪後新增）

-- 刪除舊明細
DELETE FROM SFT_BOMMF WHERE MF001 = :MF001 AND MF002 = :MF002

    :MF001, :MF002, :MF003, :MF004, :MF005, :MF006, :MF007, :MF008,
    {MF009}, {MF010}, :MF011, :MF012, :MF013, :MF014, :MF015, 1,
    :MF017, :MF018, :MF019, :MF020, 'N', :MF022, :MF023, {MF024},
